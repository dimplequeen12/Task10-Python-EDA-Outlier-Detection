# Task10-Python-EDA-Outlier-Detection
Objective:
To perform exploratory data analysis and outlier detection on a housing dataset.

Steps Performed:
- Loaded and inspected the dataset
- Generated descriptive statistics
- Checked missing values
- Visualized distributions using histogram and boxplot
- Detected outliers using IQR method
- Created outlier flag and handled outliers by capping
- Performed correlation analysis
- Exported cleaned dataset

Tools Used:
Python, pandas, numpy, matplotlib

Outcome:
Improved understanding of data patterns, outliers, and feature relationships.
